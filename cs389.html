CS389 Group 7

Group 7 Consists of myself, Dalton Haselgrove, Maximillian Pichardo and Jeffrey Sutherland. Our project will consist of a website, hosted on Willamette's servers, that will allow users to try out different types of web-based exploits such as SQLi and basic vulnerabilities. The objective of this project is to create an educational tool to be used for people interested in learning cybersec techniques.
